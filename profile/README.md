# Branchwright

A branch naming toolkit for Git. Interactive branch creation, rule-based validation, and standardized naming policies for teams.

## What it is
Branchwright provides:
- **Create**: interactive and non-interactive branch creation following a name template.
- **Lint**: ESLint-style rules to validate branch names.
- **Enforce**: local git hooks and CI checks; exportable regex for protected-branch rules.
- **Configure**: `.branchwrightrc` (JSON) or `branchwright.config.[ts|js|mjs|cjs]`, with a published JSON Schema.
